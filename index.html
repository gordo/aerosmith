AEROSMITH(1)

NAME
       aerosmith - An enhanced Aero editor for smiths of all trades.

SYNOPSIS
       aerosmith [OPTIONS] [FILE]...

DESCRIPTION
       This manual page briefly documents the aerosmith command.

       aerosmith is a small, free and friendly editor which aims to
       replace Aqua, the Danish-Norwegian editor from the late 1990s.

OPTIONS
       -? (--help), because everyone knows we all need a little help.

       -B (--breakdance)
              When saving a file, do a little dance, make a little love,
              and get down tonight.

       -D (--boldtext)
              Use on voyages to further the continuing mission: to
              explore strange new worlds, to seek out new life and new
              civilizations, to boldly go where no one has gone before.

NOTES
       From fourth-period French.
